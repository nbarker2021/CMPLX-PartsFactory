# CMPLX Memory Review

## Purpose

Prevent blind work in the CMPLX/CQE/PartsFactory corpus. Names are addresses, not essence. Every serious task starts by checking the current memory layer and translating observed names into concepts.

## Trigger

Use this skill for:

- CMPLX/CQE/Manny/TMN lineage or identity review
- historical-build reading
- merge/unification work
- code edits that may absorb older variants
- claims that a capability, tool, or concept does not exist
- any task expected to require more than two tool steps

## Output

- what was read
- what concept is involved
- names/variants observed
- current PartsFactory slot or gap
- uncertainty and next smallest reads
- whether a checkpoint/register was updated
