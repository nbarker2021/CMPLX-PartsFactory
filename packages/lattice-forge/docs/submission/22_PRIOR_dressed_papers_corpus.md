The dressed-papers corpus is a collection of whitepapers and research documents developed by the author across multiple project iterations (CMPLX, CMPLX-1T, CMPLX-Manny, Aletheia, CQE, Morphonic, MDHG, MMDB, etc.). ## The corpus principle

This is recorded in the author's working memory as a methodological principle. Names shift between system implementations; the underlying concept persists. Don't ask "which is canonical"; ask "what does each context need."
